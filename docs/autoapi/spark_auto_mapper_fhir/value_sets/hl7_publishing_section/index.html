Description: Represents the HL7 V3 publishing section that deals with the administration and management of health care activities and organizations.

UsageNote: V3 Specifications are published in a set

of “domains”, which contain interactions and related specifications for a single area of health care within which can be supported by a single, coherent set of interoperability specifications.
