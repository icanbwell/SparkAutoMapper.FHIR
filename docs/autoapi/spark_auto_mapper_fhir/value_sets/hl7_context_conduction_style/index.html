Conduction_indicator_based¶

Definition: Context conduction is not explicitly defined. The recipient of an instance must infer conduction based on the semantics of the model and what is deemed “reasonable”.

UsageNotes: Because this approach can lead to

variation in instance interpretation, its use is discouraged. From: http://terminology.hl7.org/CodeSystem/v3-HL7ContextConductionStyle in v3-codesystems.xml
