Common Languages (Written) From: http://hl7.org/fhir/ValueSet/written-language in valuesets.xml

This value set includes common codes from BCP-47

(http://tools.ietf.org/html/bcp47) for the purpose of writing; this value set (unlike the common languages value set) doesn’t include dialects except where they are relevant for written languages
