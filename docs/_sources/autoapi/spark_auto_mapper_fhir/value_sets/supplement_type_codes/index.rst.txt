   SupplementTypeCodes
   From: http://hl7.org/fhir/ValueSet/supplement-type in valuesets.xml
       SupplementType :  Codes for nutritional supplements to be provided to the
   patient. This value set is composed of SNOMED CT (US Extension) Concepts from
   SCTID 470581016 (Enteral+supplement feeds hierarchy (product)) and is provided
   as a suggestive example.
